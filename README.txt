It is Android Studio project.

The application has implemented AES 128bits encryption to encrypt or decrypt given input using given key. If the key length must be at least 16 chars in order to process the encryption. Also, all input expected to be ASCII chars.

In app/src/main/java/com/example/ganzorigbattur/encrypttext folder, you can find the implementation of AES a long with code for activities.

In app/src/main/res/layout folder, you can find the layout xml files for activities.
